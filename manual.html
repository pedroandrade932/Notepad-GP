Manual GP Notepad.

This manual is intended to answer questions about how to use the GP Notepad application.
Some information about the features of the program will be provided below.

  1. Files:

      New files.

      You can create new files of any text type in File > New File.

      Save files.

      Save your documents to your computer by going to File > Save File.
      Inside just put the file name, press save and your document will be saved.

      Open files.

      Open the documents you've already created from Files > Open File.

      Save as.

      Save a copy of your document by going to File > Save As.
